# AI 证件照小程序

一张自拍，多种专业风格。基于微信小程序 + 混元生图 AI 的职业形象照生成工具。

## 项目结构

```
├── miniprogram/           # 小程序前端
│   ├── pages/
│   │   ├── index/         # 首页 - 场景选择入口
│   │   ├── scene/         # 场景选择页
│   │   ├── upload/        # 照片上传页
│   │   └── result/        # 结果展示页 - 风格切换/付费解锁
│   ├── app.js
│   ├── app.json
│   └── app.wxss
├── cloudfunctions/        # 云函数（后端）
│   └── generatePhoto/     # AI 生图异步任务
├── docs/                  # 架构说明、接口规范
├── project.config.json    # 小程序项目配置
└── README.md
```

## 技术栈

- **前端**: 微信小程序原生开发
- **后端**: 微信云开发（云函数 + 云数据库 + 云存储）
- **AI**: 腾讯混元文生图 API
- **变现**: 微信激励视频广告 + 会员订阅
